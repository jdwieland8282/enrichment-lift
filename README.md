# Enrichment Lift Demo

This repository hosts the Enrichment Lift demo.

## Deployment Instructions

A clean deploy script ensures that only the necessary files are published to GitHub Pages.

### To deploy:

1. Make sure you are on the `main` branch.
2. Run the deploy script:

```bash
npm run deploy
```
